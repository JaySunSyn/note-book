# note-book
Gives an Element a notebook style background.

Example

 ```
    .input {
      --paper-input-container-input: {
        line-height: 25px;
      };
    }

    ...

    <note-book>
      <paper-textarea class="input"></paper-textarea>
    </note-book>
 ```
